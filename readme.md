# Bajaj OA1

Full Stack Engineering Challenge submission.

## Tech Stack
- Node.js
- Express.js
- React
- Vite
- Render

## Features
- Tree construction
- Cycle detection
- Duplicate edge detection
- Invalid input detection
- Depth calculation

## API Endpoint
POST `/bfhl`

Example request:

```json
{
  "data": ["A->B", "A->C", "B->D"]
}
```

## Run Locally

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
